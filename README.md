# json-refiner
Open your repository on GitHub.com.

Click the "Add a README" button located below your file list.

Type your project name and a short description into the editor.

Click "Commit changes" at the top right to save it.

Using your Computer
Create a new text file in your project folder.

Rename it exactly to README.md.

Write your project details inside the file and save it.

Upload/Sync the file to your GitHub account.
